# Neovim configuration

My personal configuration.
Keymaps appear with a press of space bar.

Plugins:

Coding:
- blink.cmp: auto completion.
- conform.nvim: code formatting.
- fidget.nvim: lsp status updates.
- gitsigns.nvim: git signs beside line numbers.
- LuaSnip: code snippets.
- nvim-lspconfig: LSPs.
- nvim-treesitter: code highlights.
- nvim-web-devicons: nerd font icons.
- trouble.nvim: jump through problems in code base.

Package manager for LSPs, debugger adapters, linters and formatters:
- mason-lspconfig.nvim 
- mason-tool-installer.nvim
- mason.nvim

Debugger:
- nvim-nio
- mason-nvim-dap.nvim
- nvim-dap
- nvim-dap-go
- nvim-dap-ui

Themes:
- catppuccin nvim
- gruvbox.nvim
- kanagawa.nvim
- mini.nvim defaults
- tokyonight.nvim
- rose-pine neovim
- transparent.nvim: make most themes match terminal colors.

Collections:
- snacks.nvim
- mini.nvim

Misc:
- which-key.nvim: show keymaps.
- persistence.nvim: save sessions.
- obsidian.nvim: obsidian jumping around.
- todo-comments.nvim: highlight key words in comments.
